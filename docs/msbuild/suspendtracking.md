# SuspendTracking
Suspends tracking in the current context.  
  
## Syntax  
  
```cpp 
HRESULT WINAPI SuspendTracking(void);  
```  
  
## Return value  
 An **HRESULT** with the **SUCCEEDED** bit set if tracking was suspended.  
  
## Requirements  
 **Header:** *FileTracker.h*
